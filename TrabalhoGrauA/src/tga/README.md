Programa pronto. Foi testado com diferentes arquivos, de diferentes tamanhos, nenhum deles excedendo o limite de 4096 palavras e/ou linhas.

O programa  busca no diretório do sistema operacional através do JFileChooser.

Durante os testes de cada classe foram adicionados Prints dos respectivos arrays e testados com palavras conhecidas no array para certificar se o programa estava retornando os valores requisitados.
